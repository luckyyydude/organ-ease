# OrganEase - Complete Organ Procurement System

## Overview
OrganEase is a digital platform that streamlines the organ procurement process by connecting hospitals with procurement centers in real-time, eliminating manual processes and reducing critical time delays in life-saving operations.

## Features

### ✅ Core Features
1. **Real-Time Organ Inventory** - View all available organs across multiple centers
2. **Smart Filtering** - Filter by organ type, blood group, and procurement center
3. **Organ Request System** - Multi-step form for requesting organs
4. **Blood Group Compatibility** - Automated matching based on blood types
5. **Secure Payment** - Token payment system for confirmation
6. **Request Tracking** - Track status of all organ requests
7. **Blood Camp Locator** - Find nearby blood donation camps

### ✅ Pages Included
- `index.html` - Homepage with introduction and features
- `dashboard.html` - Available organs inventory
- `requested-organs.html` - Track all organ requests
- `request.html` - Organ request form
- `blood-camp.html` - Find blood donation camps
- `payment.html` - Payment confirmation
- `confirmation.html` - Procurement center confirmation interface
